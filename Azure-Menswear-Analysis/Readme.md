# Azure Menswear Analysis

## Overview
This project focuses on understanding how Microsoft Azure can be used as a data source, backend server, and database for analytics workflows. The project explores integrating Azure services with Power BI using a menswear (T-shirt) sales dataset as the analytical context.

The primary goal of this project is to learn cloud-based data handling and BI integration rather than deep analytical modeling.

---

## Problem Statement
Modern analytics systems increasingly rely on cloud platforms for scalable data storage and processing.  
This project aims to understand how Azure services can be used to host data and connect seamlessly with business intelligence tools like Power BI for analysis and visualization.

---

## Learning Objectives
- Understand Azure as a cloud data platform
- Learn how databases and services are managed in Azure
- Explore connecting Azure data sources to Power BI
- Understand real-world cloud-to-BI analytics workflows

---

## Dataset
- Domain: Menswear (T-shirt sales)
- Data type: Structured sales and product data
- Usage: Demonstration and learning purposes

> Note: Full hands-on deployment was limited due to Azure free-tier access constraints.

---

## Tech Stack
- Microsoft Azure (Data Source & Database Concepts)
- Power BI
- SQL (Conceptual usage)
- Cloud-based analytics workflow

---

## Key Learnings
- How Azure can act as a centralized data source for analytics
- Understanding Azure servers, databases, and resource management
- Connecting cloud-hosted data to Power BI
- Basic sales analysis and visualization approaches for retail data
- Practical understanding of cloud-based BI architecture

---

## Analysis Scope
- Basic sales trend analysis
- Category-wise and product-level insights
- High-level visualization in Power BI
- Focus on integration rather than advanced analytics

---

## Limitations
- Full practical implementation was limited due to payment method restrictions for Azure free-tier access
- Project emphasizes conceptual understanding and integration workflows
