# CarePlus Support Analytics Pipeline

## End-to-End AWS Data Engineering Project

This project demonstrates the design and implementation of a cloud-native data engineering pipeline on AWS for processing customer support tickets and backend support logs.

### Key Highlights

* Incremental Data Ingestion
* Amazon S3 Data Lake Architecture
* AWS Lambda Transformations
* AWS Glue ETL Processing
* AWS Athena Validation
* Amazon Redshift Serverless Warehouse
* Event-Driven Automation
* Power BI Reporting Dashboard

### Technologies Used

* Python
* SQL
* Amazon S3
* AWS Lambda
* AWS Glue
* AWS Athena
* Amazon Redshift Serverless
* AWS IAM
* Amazon CloudWatch
* Power BI
