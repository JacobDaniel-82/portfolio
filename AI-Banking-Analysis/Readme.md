## Overview

---

## Problem Statement
Banking analytics involves complex datasets, multiple KPIs, and structured cleaning workflows.  
This project explores how AI models can be effectively used to accelerate data generation, cleaning, and visualization when guided by clear analytical requirements.

---

## Approach & Workflow
1. Generated a synthetic banking dataset using AI-generated SQL queries  
2. Loaded unclean data into SQL Server  
3. Cleaned and transformed data within SQL Server using AI-suggested SQL queries  
4. Defined business requirements and analytical goals via prompts  
5. Designed KPIs and visualizations based on AI-assisted recommendations  
6. Created DAX measures using AI guidance and validated results
