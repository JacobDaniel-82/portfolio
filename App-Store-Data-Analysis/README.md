# App Store Analytics using Python

## Overview
This project performs exploratory data analysis (EDA) on mobile app store data to uncover trends related to app categories, ratings, installs, pricing, and popularity. The analysis helps understand what factors contribute to app success in competitive app marketplaces.

---

## Problem Statement
With millions of apps available on app stores, identifying patterns behind high-performing apps is challenging.  
This project analyzes app store data to extract insights that can help developers and analysts understand user preferences, monetization strategies, and category-wise performance.

---

## Dataset
- Source: Public App Store / Google Play dataset (Kaggle)
- Approx. 10,000+ app records
- Key features include:
  - App Name
  - Category
  - Rating
  - Reviews
  - Installs
  - Price
  - Type (Free / Paid)

---

## Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## Key Analysis & Insights
- Distribution of apps across different categories
- Comparison of free vs paid apps based on ratings and installs
- Identification of top-performing categories by average rating
- Analysis of relationship between app size, pricing, and popularity
- Data cleaning and handling of missing or inconsistent values

---

## Outputs
- Cleaned and preprocessed dataset
- Exploratory data visualizations
- Insights useful for app developers and product analysts

---

## How to Run
1. Clone the repository
