# British Airways Review Analysis (Tableau)

## Overview
This project analyzes customer reviews of British Airways to understand passenger satisfaction across different service and travel dimensions. Using Tableau, the project focuses on visualizing trends in customer sentiment, ratings, and review patterns to uncover insights into overall airline performance.

---

## Problem Statement
Airlines receive thousands of customer reviews across multiple platforms, making it difficult to manually identify satisfaction drivers and recurring issues.  
This project aims to analyze British Airways customer reviews and present insights through interactive Tableau dashboards.

---

## Dataset
- Source: [dataset](./ba_reviews.csv)
- Data type: Customer reviews and ratings
- Time range: Multiple years of review data
- Key attributes:
  - Review text
  - Overall rating
  - Travel class
  - Aircraft type
  - Route / Region
  - Recommendation status

---

## Tech Stack
- Tableau

## Key Analysis & Insights
- Overall customer satisfaction trends over time
- Rating comparison across travel classes
- Analysis of service quality based on aircraft and routes
- Identification of frequent positive and negative feedback themes
- Recommendation vs non-recommendation analysis

---

## Dashboard Features
- Interactive filters for class, aircraft, and travel type
- Visual comparison of ratings and satisfaction levels
- Trend analysis using time-based charts
- Summary KPIs for quick performance overview

## Outputs
- Interactive Tableau dashboard
- Visual insights into customer experience and service quality
- Actionable observations for airline service improvement
