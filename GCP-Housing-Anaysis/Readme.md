# GCP Housing Analysis

## Overview
This project focuses on understanding how Google Cloud Platform (GCP) can be used as a data source for analytics and business intelligence workflows. Using a housing market dataset, the project explores how data can be accessed from BigQuery and connected to Power BI for market analysis and visualization.

The project emphasizes learning cloud-based data integration rather than full-scale deployment.

---

## Problem Statement
Real estate analytics often rely on large datasets that are best handled using cloud data platforms.  
This project aims to understand how housing market data stored in GCP can be queried and visualized using Power BI to generate market and sales performance insights.

---

## Learning Objectives
- Understand GCP as a cloud data platform
- Learn BigQuery data structure and querying concepts
- Explore integration between BigQuery and Power BI
- Understand cloud-based analytics workflows

---

## Dataset
- Domain: Housing market data
- Data type: Structured sales and market data
- Usage: Learning and demonstration purposes

> Note: Full hands-on deployment was limited due to GCP free-tier access constraints.

## Tech Stack

## Key Learnings
