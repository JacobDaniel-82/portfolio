# Loan Default Analysis (Power BI)

## Overview
This project analyzes loan default data to identify risk patterns and applicant characteristics using Power BI. The dataset is ingested through a dataflow gateway, cleaned and transformed using Power Query Editor, and visualized through interactive dashboards to support credit risk analysis.

---

## Problem Statement
Financial institutions need to understand loan default behavior to manage risk and improve lending decisions.  
This project aims to analyze loan performance, applicant demographics, and financial risk metrics to provide a structured view of default trends.

---

## Data Source
- Source Type: Dataflow Gateway
- Data Nature: Structured loan and applicant data
- Integration: Connected to Power BI for transformation and visualization

---

## Tech Stack
- Power BI
- Dataflow Gateway
- Power Query Editor (PQE)
- DAX
- Data Cleaning & Transformation

---

## Dashboard Structure
### Page 1: Loan Default Overview
- Overall loan and default metrics
- Default rate trends
- High-level risk indicators

### Page 2: Applicant Demographics & Profile
- Applicant age, gender, and income distribution
- Employment and profile-based insights
- Demographic patterns related to defaults

### Page 3: Financial Risk Metrics
- Year-to-Date (YTD) analysis
- Year-over-Year (YoY) trends
- Financial performance and risk indicators

---

## Key Insights
- Identification of high-risk applicant segments
- Trends in loan defaults over time
- Relationship between applicant profiles and default behavior
- Financial risk patterns using YTD and YoY metrics

---

## Outputs
- Interactive multi-page Power BI dashboard
- Risk-focused insights for lending decisions
- Structured analysis of loan and applicant data
